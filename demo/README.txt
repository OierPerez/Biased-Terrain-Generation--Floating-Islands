Project Name: Floating Island Generation

Controls:

WASD: Move the camera 
E: Zoom out
Q: Zoom in
LSHIFT: x2 speed for the controls

ImGUI display:

The Reset button will remake the current mode generating different islands.

The drop down list allows the user to select between the different modes.  By default the demo starts at the final state.

The rest of the parameters change the shape of the island, number of islands, etc.
